# Asset Manifest

The following files have been generated and prepared for the project. The LLM agent should use these relative paths in the frontend code.

## 3D Models
The models are stored as binary `.glb` files and should be loaded using `@react-three/drei`'s `useGLTF` hook.
- `/models/ac_unit.glb`
- `/models/car.glb`
- `/models/burger.glb`
- `/models/avocado.glb`
- `/models/duck.glb`
- `/models/waterbottle.glb`

## 2D Mascot Images
The mascot images are generated 2D web assets and should be used in standard React `<img>` tags or Framer Motion `<motion.img>` tags.
- `/images/earth_happy.webp`
- `/images/earth_sweating.webp`
- `/images/earth_thinking.webp`
